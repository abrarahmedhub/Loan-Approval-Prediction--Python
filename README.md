# 🏦 Loan Approval Prediction using Machine Learning

A machine learning project that predicts whether a loan application will be approved or rejected based on applicant details such as income, credit history, employment status, and other relevant features.

---

## 🚀 Project Overview

Loan approval is a critical process in the financial industry. This project aims to automate and improve the decision-making process by building a predictive machine learning model.

The model analyzes historical loan application data and identifies patterns that influence approval decisions. Based on these patterns, it predicts whether a new loan application should be approved.

---

## 🎯 Objectives

* Build a classification model to predict loan approval status
* Perform data preprocessing and feature engineering
* Analyze important factors affecting loan approval
* Evaluate model performance using appropriate metrics

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/abrarahmedhub/loan-approval-prediction-python.git
```

2. Navigate to the project folder:

```bash
cd loan-approval-ml
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Launch Jupyter Notebook:

```bash
jupyter notebook
```

5. Open and run:

```bash
untouched.ipynb
```
